How much CSS should I know before learning Sass?

Sass is really just a way to write CSS. You can certainly learn both at the same time, if you write in the SCSS format, but learning CSS is about learning how to style, layout and animate your content, learning Sass is about doing it efficiently on the production side.

Learning Sass won't do you any good if you don't know how a float or a z-index works. ...

Knowing CSS means you know how to do the things you've learned, like media queries, animations and transitions as well as things you'll need to know in the future, like flex-box and CSS grid. It also means knowing how to fix common problems and have a good grip on browser support. And many other style-related issues like how to set your typography and color schemes consistently.
